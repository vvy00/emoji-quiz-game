# 🎮 Emoji Quiz Game 🎮

A fun and interactive emoji quiz game where players guess movies, animals, food, or words from emoji clues! Built with HTML, CSS, and JavaScript.

## 🕹️ Features
- Multiple categories: Movies & TV, Animals, Food, Guess the Word
- Creative emoji puzzles (e.g., 🧈🪰 for "Butterfly")
- Score tracking
- Subtle confetti on correct answers
- Responsive design with bounce animations

## 💻 How to Play
1. Open `index.html` in your browser.
2. Choose a category.
3. Guess the word by clicking one of the emoji-based options.
4. See your score and try to get them all correct!

## 🛠️ Technologies
- HTML
- CSS (with gradient background, animations, and responsive layout)
- JavaScript (game logic and confetti effects)
