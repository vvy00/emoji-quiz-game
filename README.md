# 🎮 Emoji Quiz Game 🎮

A emoji quiz game where players guess movies, animals, food, or words from emojis. Built with HTML, CSS, and JavaScript.

## Features
- Multiple categories: Movies & TV, Animals, Food, Guess the Word
- Creative emoji puzzles (Ex. 🧈🪰 for "Butterfly")
- Score tracking
- Subtle confetti on correct answers
- Responsive design with bounce animations

## Technologies
- HTML
- CSS (with gradient background, animations, and responsive layout)
- JavaScript (game logic and confetti effects)

## AI Usage
- Used ChatGPT for debugging
